16.3 신경망 기계 번역을 위한 인코더-디코더 네트워크¶

영어 문장을 프랑스어로 번역하는 과정이다.

과정은 다음과 같다

  1. 영어문장을 뒤집어줌 : I drink milk -> milk drink I
  2. 각 단어 별 숫자 맵핑 : milk drink I -> 288 3335 72
  3. 임베딩 층을 거쳐 인코더에 입력
  4. 인코더 결과값 출력 : Je bois du lait
  5. 각 단어 별 숫자 맵핑 : Je bois du lait -> 0 51 2132 21 431
  6. 임베딩 층을 거쳐 인코더에 입력
  7. 결과값의 단어별 점수 출력
  8. 소프트 맥스층이 점수를 확률로 변환
  9. 예측 : Je bois le lait / 타깃 : Je bois du lait

분류작업과 매우 유사하기 때문에 손실함수는 sparse_categorical_crossentropy 손실함수를 사용한다.

이 모델을 구현하기 위해서는 몇 가지 처리가 필요하다.

  1. 단어 버킷 사이즈별 그룹핑

    • 위의 모델은 문장의 길이(입력 시퀀스)가 동일하다고 가정한 모델이다. 하지만 실제 문장은 길이가 다르기 때문에 sequence_length 함수를 사용해 각 문장의 길이를 비슷한 길이의 버킷으로 그룹핑하고 각 버킷에 담긴 문장이 모두 동일한 길이가 되도록 패딩을 추가한다.

      ex) 'I drink milk' -> '\ \ \ milk drink I'

  2. EOS 토큰 이후 출력은 모두 무시한다.

    ex) 'Je bois du lait \ oui' 를 입력하면 가중치는 (1, 1, 1, 1, 1, 0) 가 된다


  1. 출력 어휘 사전이 방대할 경우 느려지는 문제에 대한 처리

    • 만약 프랑스어 단어 사전에 50000개의 단어가 있다면 디코더는 50000 차원의 벡터를 출력하게 되어 매우 느려짐.
    • 디코더가 더 작은 벡터 (예 : 1,000 차원 벡터)를 출력하도록 한 다음 샘플링 기술을 사용하여 손실을 추정.
    • 따라서 대상 어휘의 모든 단어를 계산할 필요가 없다.
    • Softmax 기술 샘플링은 Sébastien Jean 등이 2015 년 에 제안
    • TensorFlow에서 sampled_softmax_loss () 함수를 통해 사용할 수 있다.

# tensorflow_addons 프로젝트는 여러가지 시퀀스-투-시퀀스 도구를 가지고 있어
# 프로덕트 레벨의 인코더-디코더를 손쉽게 만들 수 있다.

import tensorflow_addons as tfa

encoder_inputs = keras.layers.Input(shape=[None], dtype=np.int32)
decoder_inputs = keras.layers.Input(shape=[None], dtype=np.int32)
sequence_lengths = keras.layers.Input(shape=[], dtype=np.int32)

embeddings = keras.layers.Embedding(vocab_size, embed_size)
encoder_embeddings = embeddings(encoder_inputs)
decoder_embeddings = embeddings(decoder_inputs)

# LSTM 층 생성 시 최종 은닉 상태를 디코더로 보내기 위해 return_state=True로 설정
encoder = keras.layers.LSTM(512, return_state=True)
encoder_outputs, state_h, state_c = encoder(encoder_embeddings)
encoder_state = [state_h, state_c]

'''
TrainingSampler
각 스텝에서 디코더에게 이전 스텝의 출력이 무엇인지 알려준다. 
추론 시에는 실제로 출력되는 토큰의 임베딩 되고,
훈련 시에는 이전 타겟 토큰의 임베딩이 되어야한다

※ 실전에서는 이전 타임 스텝의 타깃의 임베딩을 사용해 훈련을 시작하고, 이전 스텝에서 출력된
임베딩으로 점차 바꾸는 것이 좋다
'''

sampler = tfa.seq2seq.sampler.TrainingSampler()

decoder_cell = keras.layers.LSTMCell(512)
output_layer = keras.layers.Dense(vocab_size)
decoder = tfa.seq2seq.basic_decoder.BasicDecoder(decoder_cell, sampler,
                                                 output_layer=output_layer)
final_outputs, final_state, final_sequence_lengths = decoder(
    decoder_embeddings, initial_state=encoder_state,
    sequence_length=sequence_lengths)
Y_proba = tf.nn.softmax(final_outputs.rnn_output)

model = keras.models.Model(
    inputs=[encoder_inputs, decoder_inputs, sequence_lengths],
    outputs=[Y_proba])
